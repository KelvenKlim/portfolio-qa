# portfolio-qa

Este repositório contém uma aplicação front-end construída com Vite, React, TypeScript e Tailwind CSS. O layout e os componentes seguem a estrutura shadcn-ui e Radix UI.

---

## Português (PT-BR)

### Visão geral

Projeto de portfólio com foco em componentes modernos e responsivos. Ideal para usar como base ou referência para sites pessoais e landing pages.

### Tecnologias principais

- Vite
- React + TypeScript
- Tailwind CSS
- shadcn-ui / Radix UI

### Requisitos

- Node.js (recomendado: versão LTS atual)
- npm ou yarn

### Como rodar localmente

1. Instale as dependências:

```bash
npm install
```

2. Rode o servidor de desenvolvimento:

```bash
npm run dev
```

3. Abra http://localhost:5173 (padrão do Vite)

### Scripts úteis

- `npm run dev` — inicia o servidor de desenvolvimento (Vite)
- `npm run build` — constrói a aplicação para produção
- `npm run build:dev` — build em modo development
- `npm run preview` — pré-visualiza o build de produção localmente
- `npm run lint` — roda o ESLint
- `npm run test` — executa os testes com Vitest (run)
- `npm run test:watch` — executa os testes em modo interativo/watch

> Os scripts acima estão definidos em `package.json`.

### Estrutura principal (resumida)

- `src/` — código fonte da aplicação
  - `components/` — componentes UI e seções da landing
  - `pages/` — rotas/páginas
  - `assets/` — imagens e recursos estáticos

### Contribuição

1. Faça um fork ou crie uma branch a partir de `main`.
2. Abra um Pull Request descrevendo a mudança.
